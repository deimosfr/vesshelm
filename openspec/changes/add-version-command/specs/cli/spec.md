# Spec Delta: Version Command

## ADDED Requirements

### Version Display
The CLI MUST provide a command to display its version.

#### Scenario: Version Command
When `vesshelm version` is executed:
- It MUST print the application name and version (e.g., `vesshelm 1.0.0`) to stdout.

## MODIFIED Requirements

### Application Version
- The application version MUST be `1.0.0`.
